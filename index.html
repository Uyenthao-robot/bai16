CSS Tutorial - Định dạng khung và viền

1. Ẩn phần tử HTML

Để ẩn một phần tử HTML trên trang web, bạn có thể sử dụng thuộc tính CSS display: none;.

2. Ý nghĩa định dạng test.test_more

Bộ chọn đồng thời test.test_more áp dụng cho các phần tử có cả hai lớp test và test_more. Ví dụ:

.test.test_more {
    background-color: red;
}

Trong ví dụ này, các phần tử có lớp test và test_more sẽ có màu nền đỏ.

3. Định dạng cho các đoạn văn P1, P2, và P3

Giả sử trang web có nhiều thẻ <p>, và bạn muốn định dạng:

  • P1: Khác biệt so với các đoạn còn lại.
  • P2 và P3: Có cùng kiểu định dạng và cũng khác biệt.
  • Các đoạn khác: Định dạng giống nhau.

Cách làm:

<p class="important">Đoạn P1</p>
<p class="normal">Đoạn P2</p>
<p class="normal">Đoạn P3</p>

CSS:

.important {
    color: white;
    background-color: red;
    font-weight: bold;
}

.normal {
    color: black;
    background-color: lightblue;
    font-style: italic;
}

4. Thiết lập định dạng cho khung và viền

Bạn có thể thiết lập các viền khác nhau cho từng cạnh sử dụng:

  • border-top: Định dạng viền trên.
  • border-bottom: Định dạng viền dưới.
  • border-left: Định dạng viền trái.
  • border-right: Định dạng viền phải.

Ví dụ:

.custom-border {
    border-top: 5px solid red;
    border-bottom: 3px dashed blue;
    border-left: 4px dotted green;
    border-right: 6px double purple;
    padding: 20px;
}
Đây là ví dụ về khung với các viền khác nhau cho từng cạnh.
