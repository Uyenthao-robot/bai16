Bài 16: Định dạng khung - Kết nối tri thức

Ẩn phần tử Ý nghĩa định dạng Định dạng đoạn văn Khung và viền

Luyện tập 1: Phần tử HTML có thể ẩn đi trên trang web được không?

Lời giải: Có, phần tử HTML có thể được ẩn đi trên trang web bằng cách sử dụng lệnh CSS display: none;.

p {
    display: none;
}

Luyện tập 2: Ý nghĩa định dạng test.test_more

Ý nghĩa:

  • test.test_more: Đây là bộ chọn đồng thời áp dụng cho các phần tử có cả hai lớp test và test_more.
  • background-color: red;: Đặt màu nền của các phần tử này thành màu đỏ.
.test.test_more {
    background-color: red;
}

Vận dụng 1: Định dạng đoạn P1, P2, P3

Giả sử trang web có nhiều thẻ <p>, bạn có thể định dạng:

  • P1: Khác biệt so với các đoạn còn lại.
  • P2 và P3: Có cùng kiểu định dạng.
<p class="important">Đoạn P1</p>
<p class="normal">Đoạn P2</p>
<p class="normal">Đoạn P3</p>
.important {
    color: white;
    background-color: red;
    font-weight: bold;
}

.normal {
    color: black;
    background-color: lightblue;
    font-style: italic;
}

Vận dụng 2: Thiết lập định dạng cho các khung và viền

Bạn có thể thiết lập viền khác nhau cho từng cạnh của khung bằng các thuộc tính border-top, border-bottom, border-left, và border-right.

.custom-border {
    border-top: 5px solid red;
    border-bottom: 3px dashed blue;
    border-left: 4px dotted green;
    border-right: 6px double purple;
    padding: 20px;
}
